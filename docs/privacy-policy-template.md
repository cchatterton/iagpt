# Privacy Policy Template

Public GPTs with Actions need a valid privacy policy URL. Use this as source text for a page on the publisher's website, then replace bracketed placeholders.

```text
# Analytics Chat for WordPress Privacy Policy

Last updated: [DATE]

Analytics Chat for WordPress is a GPT experience that helps interpret aggregated website analytics from a WordPress site running Independent Analytics and the Analytics Chat for WordPress plugin.

## Data Processed

When you use this GPT, it may request aggregated analytics data from the connected WordPress site, including:

- Page and post performance metrics.
- Site-level views, visitors, sessions, bounce rate, conversions, and conversion rate where available.
- Aggregated referrer and campaign metrics where available.
- Aggregated form conversion metrics where available.
- Aggregated and anonymised journey patterns where available.

## Data Not Requested

The GPT and plugin are designed not to expose:

- Raw IP addresses.
- Visitor fingerprints.
- Individual visitor browsing histories.
- WordPress user emails, names, or user IDs.
- Content editing or analytics modification capabilities.

## Public Endpoint Access

The WordPress plugin exposes public, read-only REST endpoints for aggregated analytics data. These endpoints do not require API keys and are designed not to expose personal, visitor-level, or WordPress user identity data.

## Use Of Data

Analytics data is used to answer the user's analytics questions and provide plain-English interpretation, risks, trends, and recommendations.

## Third Parties

The GPT runs in ChatGPT by OpenAI. Requests to the WordPress analytics endpoint are processed through GPT Actions. Review OpenAI's applicable privacy and data controls for ChatGPT usage.
```
